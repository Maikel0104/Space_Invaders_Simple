# SkyWalkers Invaders 👾

Developed for the Digital Games Programming course, this project focuses on recreating classic arcade games. 

Currently, the game is built using a structured programming approach. It will later be refactored into Object-Oriented Programming (OOP) to allow for greater complexity and better code organization in future updates.

`![SkyWalkers Invaders Gameplay](assets/images/gameplay.png)`

## 🎮 Controls

* **Left / Right Arrow Keys:** Move the spaceship
* **Spacebar:** Shoot
* **R:** Restart the game (on the Game Over screen)

## 🚀 Getting Started

### Prerequisites
Make sure you have Python installed on your machine.

### Installation

1. Clone the repository to your local machine.
2. Navigate to the project directory and create a virtual environment:
   ```bash 
   py -m venv venv
3. Activate the virtual environment:  
   - On Windows:
       ```bash    
       venv\Scripts\activate
   
   - On Mac/Linux:
       ```bash     
      source venv/bin/activate
4. Install Pygame:
    ```bash    
    pip install pygame
   
## ▶️ Running the Game
After installing the dependencies, you can start the game by running:

    python main.py
